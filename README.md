## 📋 项目概述

### 1.1 什么是多功能计时器？
多功能计时器是一款基于Python和PyQt5开发的桌面应用程序，提供计时器、倒计时、预设时间管理等功能。它设计简洁、功能丰富，适用于个人时间管理、工作计时、学习专注等各种场景。

### 1.2 开发背景
在快节奏的现代生活中，有效的时间管理对于个人和职业发展至关重要。无论是学生需要番茄工作法来保持专注，还是专业人士需要精确的时间跟踪，亦或是日常生活中需要定时提醒，一个可靠的计时工具都能显著提高效率。

### 1.3 目标用户群体
- **学生群体**：用于学习计时、考试模拟、休息提醒
- **办公人员**：会议计时、工作任务分配、休息提醒
- **创意工作者**：工作坊计时、创意写作时间管理
- **健身爱好者**：锻炼计时、休息间隔管理
- **家庭用户**：烹饪计时、家务时间管理、游戏时间控制
- **教育工作者**：课堂活动计时、考试时间管理
- **开发者/程序员**：编码专注时间、休息提醒

## 🚀 功能特点

### 2.1 核心功能模块

#### 2.1.1 ⏱️ 计时器功能
- **正向计时**：从0开始计时，没有上限
- **实时显示**：时、分、秒精确显示
- **进度可视化**：通过进度条展示时间流逝
- **系统托盘集成**：即使窗口最小化也能看到当前计时

#### 2.1.2 ⏰ 倒计时功能
- **灵活设置**：可设置小时、分钟、秒
- **快速预设**：内置常用时间预设（5分钟至3小时）
- **进度跟踪**：实时显示剩余时间和完成百分比
- **闹钟提醒**：时间到达时发出声音和弹窗提示

#### 2.1.3 ⭐ 预设管理
- **内置预设**：包含学习、工作常用时间设置
- **自定义预设**：用户可以创建自己的常用时间预设
- **一键启动**：点击预设即可开始对应时长的倒计时
- **最近使用记录**：自动记录最近使用的10个计时

#### 2.1.4 ⚙️ 设置系统
- **声音控制**：音量调节、静音开关、铃声选择
- **窗口设置**：窗口置顶、最小化到托盘
- **个性化设置**：支持自定义铃声文件
- **设置持久化**：所有设置自动保存

### 2.2 特色功能

#### 2.2.1 Windows任务栏进度条（专业功能）
- **进度可视化**：在Windows任务栏图标上显示进度
- **多种模式**：
  - **确定模式**：倒计时显示精确进度百分比
  - **不确定模式**：普通计时器显示动画进度
  - **暂停状态**：显示黄色暂停状态
- **窗口最小化时仍可查看进度**

#### 2.2.2 系统托盘集成
- **最小化到托盘**：程序可以在后台运行
- **托盘提示**：显示当前计时状态
- **快速操作**：双击托盘图标恢复窗口
- **托盘菜单**：提供快速操作选项

#### 2.2.3 智能提醒系统
- **多重提醒**：声音提示 + 弹窗提示
- **强制置顶**：提醒窗口保持在最上层
- **状态恢复**：提醒后自动重置计时器

## 🔧 安装与运行

### 4.1 安装方法

#### 方法一：直接运行源码（适合开发者）
```bash
# 1. 确保已安装Python 3.6+
python --version

# 2. 安装PyQt5
pip install PyQt5

# 3. 下载源代码
git clone https://github.com/duma520/SimpleTimer.git
# 或直接下载SimpleTimer.py文件

# 4. 运行程序
cd SimpleTimer
python SimpleTimer.py
```

#### 方法二：创建快捷方式（适合普通用户）
```bash
# 在Windows上创建快捷方式
# 1. 右键点击SimpleTimer.py
# 2. 选择"发送到" → "桌面快捷方式"
# 3. 重命名为"多功能计时器"
# 4. 双击运行

# 注意：确保Python已正确安装并添加到PATH环境变量
```

### 4.2 首次运行设置

#### 4.2.1 初始配置步骤
1. **首次启动**：程序会自动创建配置文件
2. **检查系统托盘**：确保系统托盘可用
3. **测试声音**：在设置中点击"测试声音"按钮
4. **调整界面**：根据屏幕分辨率调整窗口大小

#### 4.2.2 配置文件位置
```
Windows: timer_settings.json（与程序同目录）
Linux/macOS: ~/.config/timer_settings.json
```

### 4.3 常见安装问题

#### 问题1：缺少PyQt5模块
```
错误信息：ModuleNotFoundError: No module named 'PyQt5'
解决方案：pip install PyQt5
```

#### 问题2：音频播放失败
```
错误信息：无法播放声音
解决方案：
1. 检查系统音量设置
2. 确保音频驱动正常工作
3. 尝试使用不同的铃声文件
```

#### 问题3：任务栏进度条不显示
```
可能原因：
1. 非Windows系统
2. 缺少PyQt5.QtWinExtras模块
3. 窗口句柄获取失败
解决方案：
检查控制台输出信息，确认系统兼容性
```

## 🔄 基本使用

### 6.1 快速开始指南

#### 场景1：简单的计时器使用
```
步骤：
1. 打开程序
2. 点击"计时器"标签页
3. 点击"开始计时"
4. 程序开始计时，进度条开始移动
5. 需要暂停时点击"暂停"
6. 完成后点击"重置"
```

#### 场景2：基本的倒计时
```
步骤：
1. 打开程序
2. 点击"倒计时"标签页
3. 设置时间（如：25分钟）
4. 点击"开始倒计时"
5. 时间到后会有声音和弹窗提醒
6. 点击提醒窗口的"确定"
```

### 6.2 详细操作步骤

#### 6.2.1 使用计时器功能

**步骤详解：**
1. **选择计时器标签页**
   - 点击顶部的"⏱️ 计时器"标签

2. **开始计时**
   - 点击绿色的"开始计时"按钮
   - 观察时间显示从"00:00:00"开始增加
   - 进度条开始缓慢移动（每60秒循环一次）

3. **暂停和继续**
   - 点击黄色的"暂停"按钮暂停计时
   - 按钮文字变为"继续"
   - 再次点击"继续"恢复计时

4. **重置计时器**
   - 点击红色的"重置"按钮
   - 时间显示重置为"00:00:00"
   - 进度条重置为0%

5. **使用最近计时列表**
   - 在"最近使用的计时器"列表中点击任意项目
   - 程序会自动切换到倒计时标签页并设置相应时间

#### 6.2.2 使用倒计时功能

**时间设置方法：**
1. **手动输入**
   - 使用小时、分钟、秒的上下箭头调节
   - 或直接输入数字

2. **快速预设**
   - 在"快速设置"区域点击预设按钮
   - 如点击"25分钟"，自动设置25分钟倒计时

**启动倒计时：**
1. **验证时间设置**
   - 确保设置的时间大于0
   - 时间显示区会显示设置的倒计时时间

2. **开始倒计时**
   - 点击"开始倒计时"按钮
   - 时间显示开始倒计时
   - 进度条从0%开始逐渐增加到100%

**倒计时状态管理：**
1. **暂停倒计时**
   - 点击"暂停"按钮暂停倒计时
   - 进度条变为黄色暂停状态

2. **继续倒计时**
   - 点击"继续"按钮继续倒计时
   - 从暂停的时间点继续

3. **重置倒计时**
   - 点击"重置"按钮
   - 时间重置为设置的时间
   - 进度条重置为0%

#### 6.2.3 使用预设功能

**使用内置预设：**
1. **选择预设标签页**
   - 点击顶部的"⭐ 预设"标签

2. **查看预设列表**
   - 每个按钮显示预设名称和对应时间
   - 如："25分钟(00:25:00)"

3. **使用预设**
   - 点击任意预设按钮
   - 程序自动切换到倒计时标签页
   - 自动设置对应时间
   - 自动开始倒计时

**创建自定义预设：**
1. **输入预设名称**
   - 在"名称"文本框中输入易于识别的名称
   - 如："番茄工作法"

2. **设置时间**
   - 在"时间(秒)"输入框中输入秒数
   - 可以直接输入或使用上下箭头
   - 如：1500秒 = 25分钟

3. **添加预设**
   - 点击"添加预设"按钮
   - 新预设会添加到预设列表中
   - 可以立即使用新预设

#### 6.2.4 管理系统设置

**声音设置：**
1. **控制静音**
   - 勾选"静音"复选框关闭所有声音
   - 取消勾选恢复声音

2. **调整音量**
   - 拖动音量滑块调节音量大小
   - 右侧显示当前音量百分比

3. **选择铃声**
   - 下拉菜单中选择"默认铃声"
   - 或选择"自定义铃声..."选择自己的声音文件
   - 支持格式：WAV、MP3、OGG等

4. **测试声音**
   - 点击"测试声音"按钮播放当前铃声
   - 确认声音设置是否正确

**窗口设置：**
1. **窗口置顶**
   - 勾选"窗口置顶"使窗口始终在最上层
   - 适合需要持续关注计时的情况

2. **最小化到托盘**
   - 勾选"最小化到托盘"
   - 当最小化窗口时程序会隐藏到系统托盘
   - 双击托盘图标恢复窗口

**重置设置：**
1. **谨慎操作**
   - 点击"重置为默认设置"按钮
   - 确认重置操作
   - 所有设置恢复为出厂默认

## 🛠️ 故障排除

### 10.1 常见问题及解决方案

#### 问题1：程序无法启动
```
症状：双击程序无反应，或快速闪退
可能原因：
1. Python未正确安装
2. PyQt5模块缺失
3. 系统兼容性问题

解决方案：
1. 检查Python安装：命令行输入 python --version
2. 安装PyQt5：pip install PyQt5
3. 查看错误日志：从命令行运行程序查看具体错误
```

#### 问题2：声音无法播放
```
症状：时间到时无声音提醒
可能原因：
1. 系统音量静音
2. 程序静音设置开启
3. 音频驱动问题
4. 铃声文件损坏

解决方案：
1. 检查系统音量设置
2. 取消程序中的"静音"选项
3. 点击"测试声音"按钮检查
4. 尝试使用不同的铃声文件
```

#### 问题3：窗口显示异常
```
症状：窗口位置、大小异常
可能原因：
1. 配置文件损坏
2. 屏幕分辨率变化
3. 多显示器配置变化

解决方案：
1. 重置窗口设置
2. 删除配置文件timer_settings.json
3. 重启程序重新生成配置
```

#### 问题4：计时不准确
```
症状：计时器比实际时间快或慢
可能原因：
1. 系统负载过高
2. 其他程序占用资源
3. 程序bug

解决方案：
1. 关闭不必要的程序
2. 重启计时器
3. 检查程序版本，更新到最新
```

#### 问题5：任务栏进度条不显示
```
症状：Windows系统下任务栏无进度显示
可能原因：
1. 系统版本不支持
2. PyQt5.QtWinExtras模块缺失
3. 窗口句柄获取失败

解决方案：
1. 确认Windows版本支持
2. 重新安装PyQt5完整版
3. 查看控制台输出获取详细信息
```
