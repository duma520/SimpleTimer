## 第一部分：基础入门篇（适合所有人）

### 1. 软件是什么？我为什么需要它？

#### 🎯 一句话概括
这是一个**电脑上的多功能计时工具**，就像你手机的计时器，但功能更强大、更适合电脑使用。

#### 🤔 谁需要这个软件？
- **学生**：做作业计时、番茄工作法、考试练习
- **上班族**：会议计时、工作专注、午休提醒
- **家庭主妇**：做饭计时、洗衣提醒、孩子学习监督
- **健身爱好者**：运动间隔计时、休息时间控制
- **教师/讲师**：课堂时间控制、考试监考
- **程序员**：代码测试计时、工作休息提醒
- **游戏玩家**：游戏时间控制、技能冷却提醒
- **任何人**：任何需要计时的场景

#### 💡 核心价值
- **免费开源**：完全免费，没有广告，没有收费
- **功能全面**：一个软件替代多个计时工具
- **使用简单**：界面直观，一看就会
- **专业可靠**：精准计时，稳定运行

### 2. 第一次使用：5分钟快速上手

#### 第1分钟：下载与安装
```
1. 从GitHub下载软件
2. 解压到任意文件夹
3. 双击运行 SimpleTimer.py
```
⚠️ **注意**：需要先安装Python 3.6+和PyQt5，如果不了解这些，可以下载打包好的版本

#### 第2分钟：认识主界面
打开软件后，你会看到：
- **顶部标题栏**：显示软件名称和版本
- **选项卡区域**：4个标签页（计时器、倒计时、预设、设置）
- **中央显示区**：大字体显示时间
- **底部状态栏**：显示当前状态

#### 第3分钟：开始第一个倒计时
```
1. 点击"倒计时"标签页
2. 设置：1分钟（0小时1分钟0秒）
3. 点击"开始倒计时"
4. 看着时间从01:00变成00:00
5. 听到"叮"的一声提示
```
🎉 **恭喜！** 你已经完成了第一次计时

#### 第4分钟：试试计时器
```
1. 点击"计时器"标签页
2. 点击"开始计时"
3. 看着时间从00:00:00开始增加
4. 可以暂停、继续、重置
```

#### 第5分钟：调整设置
```
1. 点击"设置"标签页
2. 拖动音量滑块调节音量
3. 点击"测试声音"听效果
4. 试试"窗口置顶"功能
```

## 第二部分：功能详解篇（按需求分类）

### 4. 计时器功能：从简单到专业

#### 4.1 基础使用：就像秒表一样简单
```
典型场景：记录学习时间
1. 开始学习时点击"开始计时"
2. 学习过程中可以随时查看用时
3. 休息时点击"暂停"
4. 继续学习时再点"暂停"（变继续）
5. 结束时点击"重置"归零
```

#### 4.2 进阶技巧：多场景应用
**场景A：记录总工作时间**
- 开始工作就开始计时
- 午休、开会时暂停
- 下班时查看总有效工作时间

**场景B：番茄工作法**
- 设置25分钟倒计时（工作）
- 时间到后休息5分钟
- 使用计时器记录总专注时间

**场景C：比赛/游戏计时**
- 多个选手/队伍轮换时
- 用暂停功能切换
- 用重置功能开始新一轮

#### 4.3 专业功能：无限计时与进度显示
- **无限计时**：可以一直计时，没有上限
- **60秒循环进度**：每60秒进度条循环一次，适合长时间工作
- **颜色状态指示**：
  - 蓝色：未开始/已停止
  - 绿色：运行中
  - 黄色：已暂停

### 5. 倒计时功能：精准时间管理

#### 5.1 时间设置的三种方式
**方法1：手动输入（最精确）**
```
示例：设置1小时23分钟45秒
小时：1
分钟：23
秒数：45
总秒数计算：1×3600 + 23×60 + 45 = 5025秒
```

**方法2：快速设置按钮（最快捷）**
- 5分钟、10分钟、15分钟、25分钟
- 30分钟、45分钟、1小时、2小时、3小时
- 一键设置，无需计算

**方法3：最近使用列表（最智能）**
- 自动记录最近10次使用的倒计时
- 点击即可重复使用

#### 5.2 倒计时状态管理
| 状态 | 显示颜色 | 含义 | 操作建议 |
|------|----------|------|----------|
| **未开始** | 蓝色 | 已设置时间但未开始 | 点击"开始倒计时" |
| **运行中** | 绿色 | 倒计时正在进行 | 可暂停或等待结束 |
| **已暂停** | 黄色 | 计时暂停 | 点击"暂停"继续 |
| **已完成** | 红色 | 时间到 | 自动触发提醒 |

#### 5.3 倒计时进度系统
**数学原理**：
```
进度百分比 = (总时间 - 剩余时间) ÷ 总时间 × 100%

示例：10分钟倒计时（600秒）
- 第0秒：进度0%
- 第300秒（5分钟）：进度50%
- 第600秒（10分钟）：进度100%
```

**可视化显示**：
- 进度条：直观显示剩余比例
- 数字显示：精确到秒
- 颜色变化：不同阶段不同颜色

| **10分钟** | 600秒 | 10×60 | 咖啡时间、小憩 |

| **3小时** | 10800秒 | 3×3600 | 考试、长途任务 |

#### 6.2 创建自定义预设
**步骤详解**：
1. **命名原则**：
   - 简短明了：如"瑜伽课"
   - 包含时间：如"晨跑30分钟"
   - 避免特殊字符

2. **时间设置技巧**：
   ```
   常用转换：
   1小时 = 3600秒
   30分钟 = 1800秒
   15分钟 = 900秒
   5分钟 = 300秒
   ```

3. **保存与管理**：
   - 预设保存在`timer_settings.json`
   - 最多无限制数量
   - 按钮自动排列

### 7. 提醒系统：多种通知方式

#### 7.1 五种提醒方式对比
| 方式 | 触发条件 | 适用场景 | 优点 | 缺点 |
|------|----------|----------|------|------|
| **声音提醒** | 时间到 | 所有场景 | 最传统、可靠 | 需要开启声音 |
| **窗口抖动** | 时间到+开启选项 | 需要强烈提醒 | 视觉冲击强 | 可能干扰他人 |
| **窗口闪烁** | 时间到+开启选项 | 需要持续提醒 | 持续吸引注意 | 可能眼花 |
| **任务栏闪烁** | 时间到+开启选项（仅Windows）| 最小化时提醒 | 后台运行有效 | 仅限Windows |
| **醒目对话框** | 时间到+开启选项 | 必须确认的提醒 | 强制确认 | 打断当前工作 |

#### 7.2 各提醒方式详细设置

**声音提醒**：
- 默认：系统提示音
- 自定义：支持WAV、MP3、OGG、FLAC、AAC、M4A格式
- 音量：0-100%可调
- 静音：完全无声

**窗口抖动效果**：
```
技术原理：通过快速改变窗口位置实现抖动
强度设置：1-10级（目前固定5级）
持续时间：500-5000毫秒
触发频率：倒计时结束时触发
```

**窗口闪烁效果**：
```
闪烁模式：红白交替
闪烁次数：可设置1-10次
间隔时间：100-1000毫秒
恢复机制：自动恢复原样式
```

**任务栏闪烁（Windows专属）**：
```
API调用：FlashWindow Windows API
闪烁模式：图标高亮/恢复交替
系统要求：Windows 7及以上
权限要求：不需要特殊权限
```

**醒目对话框**：
```
设计特点：
- 红色背景突出显示
- 大字体时间提醒
- 居中屏幕显示
- 需要手动点击确认
使用建议：重要事项必须确认时使用
```

### 8. 声音系统：个性化提示音

#### 8.1 支持的音频格式详解
| 格式 | 扩展名 | 特点 | 建议用途 |
|------|--------|------|----------|
| **WAV** | .wav | 无损音质，兼容性最好 | 首选格式，系统铃声 |
| **MP3** | .mp3 | 压缩格式，文件小 | 自定义音乐片段 |
| **OGG** | .ogg | 开源格式，音质好 | 开源项目优先 |
| **FLAC** | .flac | 无损压缩 | 高音质需求 |
| **AAC** | .aac | 苹果设备常用 | iPhone用户 |
| **M4A** | .m4a | AAC容器格式 | 苹果用户 |

#### 8.2 音频文件要求
**技术规格**：
- 文件大小：建议<10MB
- 持续时间：建议<30秒
- 采样率：8000-48000Hz
- 位深度：16位最佳
- 声道：单声道或立体声

**内容建议**：
- 提示音：清脆短促
- 音乐片段：高潮部分
- 语音提醒：清晰简短
- 自然声音：鸟鸣、钟声等

#### 8.3 音量控制系统
**数字音量 vs 物理音量**：
```
软件音量 (0-100%) → 系统混音器 → 物理扬声器

重要提示：
1. 软件音量0% = 完全静音
2. 系统音量独立控制
3. 耳机/音响音量单独控制
```

**音量调节建议**：

**技术实现**：
```python
# 保存窗口几何信息
geometry = window.saveGeometry()  # 二进制数据
settings['window_geometry'] = geometry.toHex().decode()

if settings['window_geometry']:

    ))
```

### 11. 任务栏进度条（Windows专属）

#### 11.1 功能概述
**什么是任务栏进度条？**
- Windows 7+引入的功能
- 在任务栏图标上显示进度
- 常见于文件复制、下载等场景

**本软件中的应用**：
- 计时器：不确定模式（旋转动画）
- 倒计时：确定模式（百分比进度）
- 暂停：黄色暂停状态
- 完成：绿色完成状态

#### 11.2 进度条状态详解
**计时器模式**：
```
模式：不确定(indeterminate)
显示：旋转的圆点动画
含义：正在进行，无固定时长
示例：记录总工作时间时使用
```

**倒计时模式**：
```
模式：确定(determinate)
显示：从左到右填充的进度条
计算：进度 = (已过时间 ÷ 总时间) × 100%
颜色：正常为蓝色，完成变绿色
```

**特殊状态**：
| 状态 | 显示效果 | 触发条件 |
|------|----------|----------|
| **暂停** | 黄色进度条 | 点击暂停按钮 |
| **完成** | 绿色100% | 时间到 |
| **错误** | 红色 | 系统错误时 |
| **隐藏** | 不显示 | 未开始时 |

#### 11.3 技术实现原理
```python
# Windows API调用
if sys.platform == 'win32' and HAS_WIN_EXTRAS:
    from PyQt5.QtWinExtras import QWinTaskbarButton, QWinTaskbarProgress
    
    # 创建进度条对象
    self.taskbar_button = QWinTaskbarButton(self)
    self.taskbar_progress = self.taskbar_button.progress()
    
    # 设置进度
    self.taskbar_progress.setRange(0, 100)
    self.taskbar_progress.setValue(50)  # 50%进度
    self.taskbar_progress.setVisible(True)
```

```python
minutes = 30
seconds = 15
total = hours*3600 + minutes*60 + seconds  # 9015秒
```

### 24. 故障排除与常见问题

#### 24.1 安装与启动问题
**问题：无法启动软件**
```
可能原因及解决方案：

   解决方案：安装Python 3.6+

   解决方案：pip install PyQt5

3. 缺少依赖库
   解决方案：pip install PyQt5 PyQt5-sip

4. 文件损坏
   解决方案：重新下载
```

**问题：启动后立即崩溃**
```
排查步骤：
1. 查看错误信息
2. 检查Python版本
3. 检查依赖版本
4. 尝试以管理员身份运行
5. 检查系统兼容性
```

#### 24.2 功能异常问题
**问题：声音无法播放**
```
排查流程：
1. 检查系统音量
2. 检查软件静音设置
3. 检查音频文件路径
4. 测试默认系统提示音
5. 检查音频格式支持
```

**问题：任务栏进度条不显示**
```
Windows特有问题：
1. 确认Windows版本（需Windows 7+）
2. 确认PyQt5版本包含QtWinExtras
3. 检查窗口是否已关联
4. 尝试重启软件
5. 检查杀毒软件拦截
```

```
优化建议：
```
